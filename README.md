# Mac setup script

My humble setup script

## Install

Download the script:

```
curl --remote-name https://github.com/tommypeps/macSetupScript/blob/master/install.sh
```

Review the script:

```
less install.sh
```

Install the script:

```
chmod +x install.sh
./install.sh
```
